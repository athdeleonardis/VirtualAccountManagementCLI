# VirtualAccountManagementCLI
This is a CLI for the library VirtualAccountManagement.
VirtualAccountManagement is currently used as a submodule.

## Requirements
1. git
2. node package manager (npm)

## Setup
1. Clone the repo
2. Run `git submodule update --init --recursive`
3. Run `npm install`

## Build
1. Run `npm run build`

## Run
1. Run `npm run main`
